# Smart Food Label Analyzer for Health Awareness

This is a mini project developed as part of a B.Tech Computer Science program.

The goal of this project is to help users understand food product labels by
analyzing ingredients and nutritional information using OCR and rule-based
health analysis.

## Project Objectives
- Extract text from food label images
- Identify ingredients and additives
- Highlight potentially harmful ingredient combinations
- Provide a simple health awareness score
